【制作使用エンジン】Unity

【制作サーバー】cakephp4 (MYSQL)

【制作人数】3人

【制作期間】1ヶ月

【バージョン管理】GitLab（SourceTree）

【ゲームジャンル】縦スクロールシューティング

【ゲーム概要】
・縦スクロール型シューティングゲームです。

・このゲームは、上から流れてくる敵や障害物を避けながら射撃をして敵を倒していくゲームです。

・プレイヤーの攻撃方法には、５種類の弾を放つ武器を所持しており、それぞれの武器を切り替えてゲームを進めていきます。

・ステージの最後にはボスが出現し、ボスを撃破するとステージクリアになります。

・ステージには３ステージを用意しており、ステージ３のボスを倒すとゲームクリアになります。

・タイトル画面のランキングには、４種類（トータルスコア、ステージ１、ステージ２、ステージ３）があります。トータルスコアは、ゲーム開始からゲームクリア、またはゲームオーバーになった時に入手したスコアの合計値になります。

・各ステージクリアごとに、コインを手に入れることが出来ます。

・タイトル画面では、プレイヤーのスキンを変更することができます。スキンを変更するには、ゲーム内で手に入れたコインを使ってガチャを回し、スキンを手に入れることによって切り替えることができます。

【操作方法】
・プレイヤーの移動操作は、画面をタップした状態で出現するバーチャルスティックを使って移動を行います。

・プレイヤーが弾を放つには、画面のタップでオート発射（画面にタップしている間は）され、弾を発射し続けます。

・プレイヤーの武器変更は、画面上にある弾アイコンをタップをする事で武器が切り替わります。

【アピールポイント・苦労した点】
・チーム制作で自分がリーダーを務めさせていただき、作業の工程・工数管理、コードレビュー、メンバーの意見をまとめての実装案の提案などいろいろ経験させていただきました。

・ソースコードをマージする時は、競合の発生（コンフリクト）が起きないようにメンバーの作業内容やソースコードを前もって確認しながらマージ作業を行っていました。
